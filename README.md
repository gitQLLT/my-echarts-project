# test-echarts
```
使用vue proxy请求了
  CSND的排行榜接口，qq音乐的排行榜，周杰伦MV接口
  npm在日、周、月、年各npm包下载次数，可以自主测试选择
echarts图标用到了多y轴，饼图与bar/line图的下钻与返回
bar图的定时数据图标切换
echarts5 中国地图及省份、市区下钻，右键返回上一级
获取全国省份面积、人口与GDP数据
定时切换tooltip显示高亮，悬浮地图停止
可以选择日期、输入包名进行npm包下载的次数
树图展示包的各版本在本周内的下载次数
```
